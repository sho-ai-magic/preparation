# あさのじゅんびマスター 🌅

子供が朝の支度を楽しく進めるためのWebアプリです。タスクカードをタップして「できた！」をめくっていき、出発時間までのカウントダウンやごほうびスタンプで、毎朝の準備を応援します。

## 主な機能

- ✅ **タスクカード** — 「はみがき」「きがえ」などのタスクをタップするとカードがめくれて完了に。タスクは自由に追加・削除・並び替えできます
- ⏰ **時計とカウントダウン** — 現在時刻と「しゅっぱつまで あと◯じかん◯ふん」を大きく表示
- 🔔 **アラーム** — 出発の30/20/10/5/0分前に音声と画面表示でお知らせ（タイミングはON/OFF切替可能）
- 🌟 **週間スタンプ** — 時間内に全部できた日はその曜日にスタンプが押され、設定した曜日を全部達成するとパーフェクト演出。スタンプは週明けに自動リセット、パーフェクト達成回数は累計されます
- 🎉 **お祝い演出** — 全タスク完了で紙吹雪と音声読み上げのお祝い
- 🈁 **かな/漢字切替** — お子さんの学年に合わせて表示を切替
- 🎨 **テーマカラー** — あお・ぴんく・みどり・おれんじの4色
- 📱 **マルチデバイス対応** — タブレット（縦・横）、スマートフォン（縦・横）、PCそれぞれに最適化されたレイアウト

## 使い方

ブラウザで開くだけで使えます。データ（タスク・スタンプ・設定）はブラウザ内（localStorage）に保存されるため、サーバーやアカウントは不要です。

ローカルで動かす場合：

```bash
git clone https://github.com/sho-ai-magic/preparation.git
cd preparation
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

`index.html` をブラウザで直接開いても動作します（要インターネット接続）。

## ファイル構成

```
preparation/
├── index.html      # 画面の構造（HTML）
├── css/
│   └── style.css   # カスタムスタイル・アニメーション・画面サイズ別調整
└── js/
    ├── config.js   # 文言辞書（かな/漢字）・テーマ・各種設定値
    ├── state.js    # データの保持とlocalStorageへの保存/読み込み
    ├── render.js   # 画面の描画（カード・スタンプ・カウントダウン等）
    └── app.js      # 時計・アラーム・ユーザー操作などの動作
```

## 技術メモ

- ビルド不要の静的サイト（GitHub Pagesでそのまま動作）
- [Tailwind CSS](https://tailwindcss.com/)（CDN）+ [Zen Maru Gothic](https://fonts.google.com/specimen/Zen+Maru+Gothic)フォント
- 効果音はWeb Audio API、音声読み上げはWeb Speech APIを使用
